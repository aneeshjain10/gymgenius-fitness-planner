# 🏋️ GymGenius — Fitness Planner

A clean and practical **fitness planner web app** that helps users generate workout plans based on their **goal, body type, and training style**.

Built with **HTML, CSS, and JavaScript** — no frameworks, no fake AI claims.

---

## 🔥 What makes this different?

Most beginner fitness apps are either:
- too generic  
- too complicated  
- or full of fake "AI" claims  

This project focuses on:
✔ simplicity  
✔ real workout structure  
✔ fun + motivation (anime inspired)

---

- Based on:

  - body type
  - goal
  - experience level
  - workout time
  - equipment
